# Customer Language Glossary

Use this when writing landing page copy, tweets, YouTube hooks, directory listings, and community replies. These are **their words**, pulled from forums and reviews — not marketing jargon.

## How they describe the problem

### Room & environment

| Phrase | Context | Example source |
|--------|---------|----------------|
| "Recorded in a cave" | Heavy reverb, distant mic feel | [Audacity Forum](https://forum.audacityteam.org/t/podcast-sounds-like-it-was-recorded-in-a-cave-is-there-a-way-to-correct-this/152715) |
| "Sounds like it was recorded in a bathroom" | Echo, tile reflections | Common across podcast forums |
| "Roomy" / "hollow" / "boxy" | Untreated room acoustics | Audacity Forum, Descript Canny |
| "Untreated room" | Home office / bedroom recording | iZotope blog, PodRewind |
| "Back of a cave… mic 10 meters further back" | Extreme distance + reverb | Audacity Forum |
| "Coffee shop recording" | Ambient chatter, espresso machines | VEED reviews, ElevenLabs content |
| "HVAC hum" / "fan noise" / "AC" | Steady-state background noise | Widespread |
| "Keyboard clicks" / "typing" | Intermittent noise AI struggles with | CapCut, LumiClip, community posts |
| "Construction outside the window" | Intermittent loud events | Descript tutorials, PodRewind |

### Audio quality failures

| Phrase | Meaning |
|--------|---------|
| "Unusable" | Audio so bad they'd re-record or abandon |
| "Borderline unusable" | Needs rescue before publishing |
| "Damage control" | Fix guest/remote audio, not polish host |
| "Salvage" / "rescue" | Save a recording that would otherwise be trashed |
| "Good enough" | Accepts imperfection over artifacts |
| "Broadcast ready" | Their bar for publishable quality |
| "Studio-quality" | Aspirational — what every tool claims |
| "Crisp and clear" | What they want after cleanup |
| "Natural-sounding" | **The #1 quality bar** — not just "clean" |

### AI processing artifacts (the anger words)

| Phrase | Emotional weight |
|--------|------------------|
| "Robotic" | High — top complaint across Adobe, Descript, Krisp, ElevenLabs |
| "Metallic" / "tinny" | High — hollow EQ + over-processing |
| "Muffled" / "dull" / "flat" | Lost sibilance, lost presence |
| "Digitized" / "compressed" | Adobe V2 specifically |
| "Over-processed" | User pushed slider too high or tool defaulted aggressive |
| "Underwater" / "warbling" | Spectral noise reduction artifacts |
| "Speaking through a long tube" | Audacity noise reduction gone wrong |
| "Elmer Fudd-like" | ElevenLabs on extreme noise |
| "Listener fatigue" | Long-form podcasts — enhancement tires ears |
| "Voice destroyer" | YouTube reviewer on Adobe V2 |
| "Unusable garbage" | Descript Studio Sound regression (Canny) |
| "Choppy" | Krisp real-time, Cleanvoice edits |
| "Ghost voices" / "other people in my silence" | Adobe V2 privacy/quality bug |

### Workflow & tool frustration

| Phrase | What it signals |
|--------|-----------------|
| "One click" / "magic button" | Desired simplicity |
| "Black box" | Automation with no sliders (Alitu) |
| "Workflow fragmentation" | Too many tools for one job |
| "Export and re-import" | Tool doesn't fit their editor |
| "No rollback" | Update made things worse |
| "Got worse after the update" | Trust broken |
| "I wish I'd known about it earlier" | Discovery problem (iZotope RX) |
| "Steep learning curve" | RX, Audacity, Auphonic |
| "Hours in Audacity" | Manual cleanup after AI fails |
| "Fire and forget" | Auphonic's appeal |

### Pricing & business resentment

| Phrase | Sentiment |
|--------|-----------|
| "A massive scam" | Krisp yearly fee |
| "Not worth it" | Subscription vs. quality delivered |
| "Credits expire" | LALAL.AI, ElevenLabs |
| "10€ in the trash" | Failed processing (Trustpilot) |
| "Paying for features I don't use" | Suite tools (Descript, Podcastle, Alitu) |
| "One-time purchase" | Strong preference — iZotope RX praised for this |
| "Free tier is too restrictive" | 30 min/day limits, watermarks |
| "Bait and switch" / billing complaints | VEED, Podcastle, LALAL.AI (Trustpilot) |

### Privacy & trust

| Phrase | Concern |
|--------|---------|
| "Uploading to the cloud" | Confidential interviews, sermons, client work |
| "No data leaves your Mac" | MetalVoice, local tools — strong appeal |
| "On-device" / "locally" | Privacy + offline |
| "Sensitive interviews" | Luma landing page already uses this framing |

## Sentence patterns to mirror

**Asking for help (Reddit/forum style):**
- "Is there any way to fix reverb in post?"
- "My podcast sounds like it was recorded in a cave — help"
- "What's the best tool for cleaning up remote guest audio?"
- "Adobe Enhance made it worse — alternatives?"
- "Studio Sound sounds robotic at default — what am I doing wrong?"
- "Is Krisp worth the subscription?"
- "Can I fix this without learning iZotope RX?"

**Venting (high engagement):**
- "Enhance Speech has somehow gotten worse"
- "Since the V2 update my audio sounds like a 16khz MP3 with the doppler effect"
- "The new studio sound upgrade is so much WORSE than the original"
- "I've had to completely switch to Adobe Podcast Enhance because [Descript] is largely unusable"
- "I spent hours trying to fix my audio manually"

**Recommendation requests:**
- "What's your stack for podcast audio cleanup?"
- "Adobe vs Descript vs Auphonic?"
- "Free alternative to Adobe Enhance?"
- "One-time purchase noise removal?"

## Words to avoid (from marketing/CONTEXT.md)

| Don't say | Say instead |
|-----------|-------------|
| User base, audience, customer | Creator-Editor |
| Funnel, campaign | Marketing Channel, Content Loop |
| Mini app | Free Tool |

## Copy angles that match their vocabulary

1. **"Fix the recording, keep the voice"** — addresses robotic/metallic fear
2. **"Your files stay on your hard drive"** — privacy without saying "enterprise-grade security"
3. **"Post-recording cleanup for Creator-Editors"** — not real-time, not a DAW, not a suite
4. **"When cloud tools change overnight, your workflow doesn't"** — update regression pain
5. **"Rescue bad audio without sounding like AI"** — salvage + naturalness
